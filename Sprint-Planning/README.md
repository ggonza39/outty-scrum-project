# Sprint Planning

This directory contains documentation and artifacts related to sprint planning and execution for the Outty project.

Each sprint has its own subfolder that includes:

- Sprint plan (forecast, selected backlog items, sprint goal, rationale)
- Sprint backlog with decomposed tasks
- Kanban board link or screenshots
- Burndown chart
- Daily Scrum documentation
- Sprint Review notes
- Sprint Retrospective notes
- Evidence of pairing or mob programming
- Deployment link for the sprint increment
